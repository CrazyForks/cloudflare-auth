# Code of Conduct

Cloudflare Auth is an independent open-source project. Participation in this
project should be professional, respectful, and focused on improving the
software and its documentation.

## Expected Behavior

- Use welcoming and clear language.
- Respect different experience levels and viewpoints.
- Keep technical disagreement focused on evidence, tradeoffs, and project
  goals.
- Report security-sensitive issues through the private process in
  [SECURITY.md](SECURITY.md).

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without explicit permission.
- Repeated disruption of maintainers, contributors, or issue discussions.
- Sharing secrets, raw auth tokens, raw cookies, full user databases, raw IP
  addresses, or raw user agents in public project channels.

## Enforcement

Maintainers may edit or remove comments, close issues or pull requests, limit
participation, or block users when behavior harms the project or its
contributors. Security disclosures remain governed by [SECURITY.md](SECURITY.md).

## Scope

This code applies to project spaces including issues, pull requests,
discussions, release artifacts, and community support channels controlled by the
maintainers.
